Overview

In my Network and Web Data technologies class, I had to do an Activity wherein, I had to build a webpage using Bootstrap and this was my first time building any kind of website! I seized this opportunity I got in order to learn more about designing webpages and also was fascinated to use and learn about Bootstrap framework. I built a simple Tourism website where I talk about my passion for travel and the different tourist attractions this World has to offer!

Tech Stack

The application was built using Bootstrap framework and Glitch was used to build and host this website. I have used several Bootstrap components in my webpage which I will talk about in more detail in the Description section below.

Timeline

October 2022

Description

This webpage was my very first hands on experience with building a website from scratch and I used Bootstrap framework since I wanted to learn more about it. Bootstrap is much more convinient when it comes to text/content allignment and I added various components to my website with great ease, so I chose this framework. Primarily, some of the components I added were:

  • Nav bar: I have added this component at the top most section of the website which consists of various options such as 'about my page', 'my trip video', 'Explore' and 'Contact me'
  • Carousel: This is at the start of the website page right after the nav bar and is a website page template consisting of a carousel of sliding images of various destinations.
  • In-page reference links: I have also linked these nav bar options to specific sections on the webpage. So, when we click on any one of the options, it directs us to that particular section of the page.
